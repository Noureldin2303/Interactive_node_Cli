## Running the CLI Tool

To start the CLI tool, run:

```sh
nooby_cli <command>
```

## Available Commands

- **`c | create`**: Create a new project.
- **`l | list`**: List all technologies.
- **`help`**: Provide help and usage information.
- **`version`**: Display the current version of the CLI tool.
